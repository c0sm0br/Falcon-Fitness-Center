The purpose of a table is to organize information. Important information about tables are listed below

  • Table tags are paired tags. Each table will need a table start and an end tag.
  • Table rows are paired tags. Each row will need a tr start and an end tag.
  • Each cell (box in the table) is a paired tag.
  • Table cells can contain text, graphics, and other HTML elements.
